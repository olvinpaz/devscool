Elemento <body>

El <body> en HTML es un elemento fundamental que contiene todo el contenido visible de una página web, incluyendo texto, imágenes, videos, enlaces, formularios, y otros elementos interactivos. Es el contenedor principal donde se renderiza todo lo que un usuario ve y con lo que interactúa en una página.

Está organizado a menudo con elementos semánticos como <header>, <main>, <footer>, <section>, <article>, y más.
