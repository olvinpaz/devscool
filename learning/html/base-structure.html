Estructura base de un documento HTML

El siguiente código define la esctructura base de un documento HTML

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Título de la página</title> </head> <body> Aquí va el contenido que se muestra al usuario </body> </html>



<!DOCTYPE html>

La declaración <!DOCTYPE html> indica al navegador que este es un documento HTML5. Aunque no es una etiqueta HTML propiamente dicha, es necesaria para asegurar que el documento se interprete correctamente.


<html>

Es el elemento raíz de un documento HTML. Es el contenedor principal que envuelve todo el contenido del documento HTML y marca el inicio del documento HTML. Todo lo que va dentro de estas etiquetas es considerado parte del documento HTML, por lo cual todo (excepto el <!DOCTYPE html>) debe estar dentro de la etiqueta <html> y </html>.

El atributo "lang" en la etiqueta <html> especifica el idioma principal del contenido de la página web. Este atributo es importante para navegadores, motores de búsqueda, y tecnologías asistivas, como lectores de pantalla, ya que les ayuda a determinar cómo procesar y mostrar el contenido de manera adecuada.

Los valores que recive el atributo "lang" pueden ser:
"en" para ingles
"es" para español
"fr" para frances
y cualquier otro idioma

¿Por qué es importante?

1. Accesibilidad: Los lectores de pantalla usan el atributo "lang" para seleccionar la pronunciación y la entonación correcta. Esto mejora la experiencia para personas con discapacidades visuales.

2. SEO (Optimización para Motores de Búsqueda): Los motores de búsqueda utilizan el atributo "lang" para entender el idioma de la página y para mostrar los resultados de búsqueda relevantes a los usuarios en el idioma correcto.

3. Renderización Correcta: Algunos caracteres especiales y símbolos pueden renderizarse de manera diferente según el idioma. Especificar el idioma ayuda a garantizar que se rendericen correctamente.

<head>

El elemento <head> contiene metadatos sobre el documento, es decir, información que no se muestra directamente en la página web, pero que es esencial para el funcionamiento correcto del sitio web, como su título, enlaces a archivos CSS, scripts, la indexación en motores de búsqueda, y la interacción con navegadores y redes sociales y otras configuraciones.

Algunos de los elementos más comunes que se encuentran dentro de un <head>:

1. <title>: Define el título del documento, que se muestra en la pestaña del navegador y como título en los resultados de búsqueda.

<title>Título de la página</title>


2. <meta>: Etiquetas que proporcionan metadatos adicionales. Algunas de las más comunes incluyen:

Charset: Define la codificación de caracteres.
<meta charset="UTF-8">

Viewport: Establece el área visible del navegador, crucial para el diseño responsivo que se adapta a las diferentes pantallas de los disposivo.
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Description: Proporciona una breve descripción del contenido de la página, útil para los motores de búsqueda.
<meta name="description" content="Una descripción breve de mi página web.">

Keywords: Palabras clave relacionadas con el contenido de la página (menos usado en la actualidad).
<meta name="keywords" content="Cursos, Tecnologías, Programación">

3. <link>: Se usa para vincular otros recursos, como hojas de estilo CSS, iconos, fuentes, etc.

<link rel="icon" href="favicon.ico"> <link rel="stylesheet" href="styles.css">

4. <style>: Permite incluir CSS directamente dentro del documento HTML. Aunque no es recomendado para grandes volúmenes de código CSS, puede ser útil para estilos rápidos.


5. <script>: Se utiliza para vincular o incluir scripts de JavaScript que afectan al comportamiento de la página.


6. <base>: Define la URL base para todos los enlaces relativos en el documento.


7. <meta property="og:title"> y otros metadatos de Open Graph: Utilizados para mejorar la forma en que se comparte la página en redes sociales, controlando cómo se muestran el título, la descripción, la imagen, etc.


8. <noscript>: Se muestra cuando el navegador del usuario no soporta o tiene deshabilitado JavaScript.


En resumen el <head>: es el lugar donde se incluyen todas las configuraciones y enlaces necesarios para que el navegador pueda interpretar, visualizar y mostrar correctamente el contenido del documento HTML. Es crucial para la optimización SEO, la apariencia de la página y su funcionalidad general.
