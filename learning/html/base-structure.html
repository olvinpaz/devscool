Estructura base de un documento HTML

El siguiente código define la estructura principal de un documento HTML.

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Título de la página</title> </head> <body> Aquí va el contenido que se muestra al usuario </body> </html>
