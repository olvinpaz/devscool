Etiquetas HTML (tags)

Antes de continuar

Si te pudiste dar cuenta, en la narrativa del cuento se puede visualizar una "imagen", un título, y un párrafo. Esto nostros lo sabemos como humanos que somos, porque así lo hemos aprendido, pero el navegador web, no lo sabe, por lo que es necesario hacerle saber, cuando es una imágen, un título, un parrafo, un botón, una lista sin orden, una lista ordenada etc.

Es aqui donde entra en acción el Lenguage de Marcado de HiperTexto HTML (HyperText Markup Language) el cual nos permite ir marcando todo el contenido de cada ducumento HTML, para que el navegador lo pueda interpretar correctamente a traves de lo que se conoce como etiquetas HTML o tags.

Las etiquetas HTML definen cómo se organiza y se presenta el contenido en el navegador. Cada etiqueta tiene un propósito específico y actúa como un contenedor que da formato, organiza o establece características para el contenido. HTML utiliza una sintaxis de apertura y cierre para definir estos elementos, que incluyen texto, imágenes, enlaces, formularios, scripts y otros tipos de contenido interactivo. La mayoría de las etiquetas son pares, con una etiqueta de apertura y una de cierre, aunque algunas, como <img> o <br>, son auto-cerrables.

Las etiquetas no solo determinan la apariencia básica del contenido, sino que también comunican la semántica del contenido al navegador y a los motores de búsqueda, mejorando la accesibilidad y el SEO. Además, HTML trabaja en conjunto con CSS (para el estilo) y JavaScript (para la interactividad), lo que permite construir páginas web modernas y dinámicas. Cada etiqueta tiene atributos que permiten personalizar aún más su comportamiento, añadiendo propiedades como clases, identificadores, estilos en línea y enlaces a scripts.
