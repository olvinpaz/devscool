Atributos en HTML

Los atributos en HTML son como adjetivos que describen y modifican a los elementos. Estos atributos proporcionan información adicional sobre el elemento, como su estilo, comportamiento o contenido.

Atributos Comunes a Muchos Elementos HTML

Estos atributos se pueden aplicar a una amplia variedad de elementos HTML para personalizar su apariencia y comportamiento:

  • id: Asigna un identificador único a un elemento. Se utiliza para seleccionar elementos con CSS o JavaScript.
  • class: Asigna una o más clases a un elemento. Las clases se utilizan para aplicar estilos comunes a múltiples elementos.
  • style: Permite definir estilos en línea para un elemento, sobreescribiendo los estilos definidos en una hoja de estilos externa.
  • title: Proporciona un texto de título para un elemento. Este texto aparece como una pequeña burbuja cuando el usuario coloca el ratón sobre el elemento.
  • lang: Especifica el idioma del contenido de un elemento.
  • dir: Especifica la dirección del texto (izquierda a derecha o derecha a izquierda).
  • data-*: Permite almacenar datos personalizados en un elemento. Estos datos pueden ser utilizados por JavaScript.
