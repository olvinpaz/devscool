Párrafo <p>

La etiqueta <p> en HTML se utiliza para definir un párrafo de texto. Es uno de los elementos más básicos y esenciales en HTML, y se utiliza para agrupar bloques de texto, proporcionando estructura y legibilidad a los contenidos dentro de una página web.

  2. Estructura de Documentos: <p> añade un margen superior e inferior por defecto, lo que ayuda a separar visualmente los párrafos unos de otros, mejorando la legibilidad.

  3. Anidamiento: No se recomienda anidar párrafos dentro de otros párrafos. Esto podría causar resultados inesperados y no es una práctica válida.

  4. Uso de Texto y Otros Elementos: Puede contener texto, enlaces, imágenes en línea, elementos en negrita <strong> en cursiva <em>, entre otros, pero no debe contener otros elementos de bloque como otro <p>, <div>, <h1>

    etc.

  5. Atributos Comunes:

    • id : Para identificar el párrafo de forma única dentro del documento.
    • class : Para aplicar estilos CSS específicos.
    • style : Para aplicar estilos en línea directamente al párrafo.
    • title : Proporciona información adicional que aparece al pasar el cursor sobre el párrafo.
